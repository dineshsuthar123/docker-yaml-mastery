## 🛠 Common Commands

### Docker Compose Essentials
```bash
# Start services
docker-compose up -d

# View logs
docker-compose logs -f [service-name]

# Stop services
docker-compose down

# Rebuild and start
docker-compose up --build

# Scale services
docker-compose up --scale web=3

# View running services
docker-compose ps

# Execute commands in running containers
docker-compose exec [service-name] [command]
```

### Useful Docker Commands
```bash
# Clean up system
docker system prune -a

# View images
docker images

# View containers
docker ps -a

# View networks
docker network ls

# View volumes
docker volume ls
```
